getFluidColor

open fun getFluidColor(view: BlockRenderView, pos: BlockPos, state: FluidState): Int
Content copied to clipboard

Get the tint color for a fluid being rendered at a given position.

Note: As of right now, our hook cannot handle setting a custom alphatint here - as such, it must be contained in the texture itself!

Return

The tint color of the fluid.

Parameters

view

The world view pertaining to the fluid. May be null!

pos

The position of the fluid in the world. May be null!

state

The current state of the fluid.
