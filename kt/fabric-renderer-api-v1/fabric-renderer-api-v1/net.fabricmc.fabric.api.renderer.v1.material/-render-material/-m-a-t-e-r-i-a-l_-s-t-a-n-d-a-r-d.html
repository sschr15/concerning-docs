This will be identical to the material that would be obtained by calling find on a new, unaltered, MaterialFinder instance. It is defined here for clarity and convenience.

Quads using this material use getBlockLayer ofthe associated block to determine texture blending, honor block color index, are non-emissive, and apply bothdiffuse and ambient occlusion shading to vertex colors.

All standard, non-fluid baked models are rendered using this material.
