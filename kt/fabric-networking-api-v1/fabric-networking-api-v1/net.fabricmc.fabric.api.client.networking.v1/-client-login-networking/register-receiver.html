registerReceiver

open fun registerReceiver(channelName: Identifier, queryHandler: ClientLoginNetworking.LoginQueryRequestHandler): Boolean
Content copied to clipboard

Registers a handler to a query request channel.

If a handler is already registered to the {@code channelName}, this method will return {@code false}, and no change will be made.Use unregisterReceiver to unregister the existing handler.

Return

false if a handler is already registered to the channel name

Parameters

channelName

the id of the channel

queryHandler

the handler

Throws

java.lang.IllegalStateException

if the client is not logging in
