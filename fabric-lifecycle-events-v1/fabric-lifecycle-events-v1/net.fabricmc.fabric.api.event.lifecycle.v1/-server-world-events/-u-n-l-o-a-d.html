UNLOAD

val UNLOAD: Event<ServerWorldEvents.Unload>
Content copied to clipboard

Called before a world is unloaded by a Minecraft server.

This typically occurs after a server has started shutting down.Mods which allow dynamic world (un)registration should use this event so mods can let go of world handles when a world is removed.
