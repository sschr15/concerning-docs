afterChangeWorld

abstract void afterChangeWorld(Entity originalEntity, Entity newEntity, ServerWorld origin, ServerWorld destination)
Content copied to clipboard

Called after an entity has been recreated at the destination when being moved to a different world.

Note this event is not called if the entity is a ServerPlayerEntity. AfterPlayerChange should be used to track when a player has changed worlds.

Parameters

originalEntity

the original entity

newEntity

the new entity at the destination

origin

the world the original entity is in

destination

the destination world the new entity is in
